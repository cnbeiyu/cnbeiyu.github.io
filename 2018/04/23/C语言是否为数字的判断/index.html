C语言是否为数字的判断

Posted on 2018-04-23 |

#
最近在完成C语言课程的作业中发现在很多程序中，如果自己有更高的要求，是经常要用到对输入字符串判断是否为纯数值，于是对近来学到的这种判断方法做一个记录与整理

使用scanf()函数的返回值

这是我初期使用最多的方法，直接利用scanf()函数的返回值来判断输入的是否为数值，如果是数值则会返回1，所以可以直接用scanf()函数的返回值来判断
但是这样的方法会存在一个问题，当输入的第一个值为数值后，scacnf()函数的返回值会直接为1，所以如果输入像“1cntwzcom”这样的字符串，scanf()函数返回的值是1，目前笔者还未能清楚地知道这种情况出现的原因，了解过后之后会回来补坑，所以单纯地这样使用scanf()函数来判断输入的字符串是否为纯数值是不严谨的。

使用string库中的isdigit()函数

isdigit()函数有一个参数,是用来判断是否为纯十进制数的字符串，如果给的参数 是十进制数字，则为 true；否则，为 false。

通过ASCII码来判断

这应该是最容易想到的一种方法，以数组的方式循环判断每一个元素是否在ASCII码48~57之间，也是最严谨的一种方法
